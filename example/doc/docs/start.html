创键一个场景

这部分主要通过示例演示在浏览器中如何加载。

在浏览器中导入 bustard

<html lang="zh">
  <head>
    <meta content="text/html; charset=utf-8" http-equiv="content-type" />
    <title>bustard</title>
    <style>
      #container {
        width: 100%;
        height: 100%;
      }
    </style>
  </head>

  <body>
    <div id="container"></div>
    <script src="js/bustard.js"></script>
    <script>
      // 代码写在这
    </script>
  </body>
</html>

创键一个场景

bustard 创键场景只需要传入你需要显示场景的 dom 元素，bustard 会自动填充整个元素显示。

var container = document.getElementById('container')
var bustard = new Bustard(container)

显示模型

当场景创键完毕后，加载模型需要调用一下 bustard 自带的 loader 插件。

var loader = bustard.use(new Bustard.Loader())
loader.gltfLoadByUrl('http://58.208.85.42:4000/403.glb')

结果

至此，第一个 bustard 程序就结束了，下面是完整代码。

<html lang="zh">
  <head>
    <meta content="text/html; charset=utf-8" http-equiv="content-type" />
    <title>bustard</title>
    <style>
      #container {
        width: 100%;
        height: 100%;
      }
    </style>
  </head>

  <body>
    <div id="container"></div>
    <script src="js/bustard.js"></script>
    <script>
      var container = document.getElementById('container')
      var bustard = new Bustard(container)
      var loader = bustard.use(new Bustard.Loader())
      loader.gltfLoadByUrl('http://58.208.85.42:4000/403.glb')
    </script>
  </body>
</html>
